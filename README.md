# guru_guru

Transforma lo que quieras al lenguaj de Guru Guru

## Instalación

Agrega la siguiente línea en tu Gemfile:

    gem 'guru_guru'

y ejecuta:

    $ bundle

O instala de forma manual:

    $ gem install guru_guru

## Uso

```ruby
require 'guru_guru'

"Hola".to_guru_guru => "hoga"

```
